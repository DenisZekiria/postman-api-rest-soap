## Използвани технологии и инструменти
Postman – дизайн, изпълнение и автоматизация на API заявки
Fiddler – HTTP/HTTPS трафик анализ и диагностика
REST API – CRUD операции и интеграционно тестване
SOAP API – тестови сценарии към външен онлайн магазин
JSON – обмен и валидация на данни

## Обхват на проекта
Проектът включва автоматизирани backend тестове за мобилно приложение, базирано на REST API, обхващащи стандартни CRUD операции, валидация на отговори и статус кодове.
Допълнително е реализирана тестова интеграция със SOAP API, използвана за проверка на комуникация с външен онлайн магазин и коректност на XML отговорите.
